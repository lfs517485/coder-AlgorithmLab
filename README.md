## 编译运行

```bash
# 编译项目
mvn compile

# 运行指定类（需要指定完整类名）
mvn exec:java -Dexec.mainClass="Hot100.链表.addTwoNumbers.Solution"
```

## 学习建议

1. **按题型攻关**：每个目录对应一种算法题型，建议集中练习
2. **先思考后看答案**：每道题先自己实现，再对比最优解法
3. **设计模式理解**：结合实际操作理解各模式的适用场景
